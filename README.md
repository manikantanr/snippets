# Start HTTP static file server

- Using Caddy webserver `caddy file-server --browse :80`
- Using Python3 `python -m http.server 80`


# Get IP address

- cmd: `ipconfig`
- powershell: `Get-NetIPAddress -AddressFamily IPV4` 
- bash: `ifconfig` or `hostname -I` or `ip a`
